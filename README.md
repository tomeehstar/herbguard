## 🧪 About HerbGuard

Millions of people use herbal remedies daily — but few know the risks of mixing them with prescription drugs.

**HerbGuard** is a clinical-grade herbal medicine safety checker that helps users and healthcare professionals identify dangerous herb-drug interactions, flag contraindications, and make informed decisions about herbal supplementation.

> *Built at the intersection of traditional medicine and modern healthcare technology.*

---

## ✨ Features

- 🔍 **Herb Safety Checker** — Search and review safety profiles of common herbs
- ⚠️ **Drug Interaction Alerts** — Get instant warnings about dangerous herb-drug combinations
- 🟢 🟡 🔴 **Risk Level Indicators** — Clear colour-coded safety ratings (Safe / Caution / Danger)
- 📋 **Patient Profile Input** — Personalised safety checks based on medications and conditions
- 📱 **Fully Responsive** — Works seamlessly on mobile and desktop
- ⚡ **Zero Dependencies** — Pure HTML, CSS and JavaScript. No frameworks. No installs.
